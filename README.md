# Cryptocurrency & GPU Price Analysis

## Project Overview
This 2021 data analysis project investigates the impact of the Chinese cryptocurrency market on global GPU (graphics card) prices.

## Key Objectives
- Analyze the relationship between Chinese market trends and cryptocurrency valuations
- Perform data cleaning and visualization on relevant market datasets
- Develop predictive insights about future GPU pricing based on crypto market movements

## Collaborators
- Daniel Isidro Hernandez
- Allighieri Monzalvo

## Project Details
This analysis was conducted during 2021, focusing on how regulatory and market shifts in China affected cryptocurrency values, with downstream effects on graphics card demand and pricing.

## Files Included
- Data cleaning scripts
- Visualization notebooks
- Analysis reports
- Raw/processed datasets (where available)

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib/Seaborn)
- Jupyter Notebook
- Statistical analysis libraries

## Outcomes
The project successfully identified correlations between Chinese market policies and GPU price fluctuations, providing actionable insights for market prediction.
